# /comfy-setup - Set up the local ComfyUI stack

Run the cross-platform setup walkthrough: detect the GPU, install/locate ComfyUI, install comfy-local-mcp, register the comfy-local + Hugging Face MCP servers, bootstrap a model that fits, and smoke-test an image.

## Process
Invoke the **comfy-local-setup** skill and follow it step by step, confirming each step with the user before proceeding.

Usage: `/comfy-setup`
